Comandos condicionais

São estruturas de controle de fluxo que permitem que o codigo tome decisões.

Elas funcionam avaliando uma expressão logica(condiçao) quye resulta verdadeiero (true) ou false

Com base nesse resultado, nosso script toma uma decisão e execulta as açõesconforme resultado

Exemplos de cenarios que envolvem logica condicional:

  • Processo de login: dados certos? entra...errados? não entra
  • \vitrine de produtos: tem estoque? mostra o produto...senãpo, não mostra!
  • Cadastro de senha segura: o valor digitado atende os vcriterios ? se sim, efetiva o cadastro... senão, não efetiva
