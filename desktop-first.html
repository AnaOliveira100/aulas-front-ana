Media Queries - Mobile first

Conceito

Media Queries são regras especiais em que criamos alterações na pagina de acordo com determinadas condiçoes (exemplo: tamanho da tela).

Essas alterações ocorrem com as configurações de breakpoints

As media queries são aplicadas de acordo com a estrategia usada para design responsivo, podendo ser;

  • mobile-first: do pequeno para o grande
  • Desktop mobile-first: do grande para o pequeno
